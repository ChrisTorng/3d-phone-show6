# 3D 手機展示應用程式

使用 Three.js 建立的互動式 3D 手機展示平台。

## 功能特色

- 互動式 3D 手機模型展示
- 支援各種視角切換和手勢操作
- 回應式設計，適應各種螢幕尺寸
- 針對行動裝置效能最佳化

## 技術堆疊

- Three.js - 3D 渲染引擎
- Tween.js - 平滑動畫效果
- 現代 JavaScript (ES6+)
- 回應式 HTML/CSS
